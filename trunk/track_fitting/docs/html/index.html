Algorithm

The main algorithm can be found in fit_track():

Given an intial x-y coordinate, image, and delta (pixel radius) fitTrack will:

  1. Mark the current pixel
  2. Obtain a sub-image of the next frame centered at the current frame's blob coordinates
  3. Calculate
  4. Assign the current pixel to the closest max pixel from the previous calculation
  5. Repeat

The algorithm state is maintained by the following variables:

  • z_curr - current frame or stack index
  • row_curr - current row index (m)
  • col_curr - current column index (n)
Parameters:
imgthree dimensional (M,N,Z) binary image, third dimension represents time
rowrow coordinate of starting pixel from img(:,:,Z)
colcolumn coordinate of starting pixel from img(:,:,Z)
z_initinitialization frame, allows the track to start at any level
delta (radius) for sub-image
Return values:
img_outA 3D image with only one track.
